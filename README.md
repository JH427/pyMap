# pyMap

Built for UGA AESL

Requires 'config.ini':

>[shares]<br />
>A = \\\\share\folder<br />
>[login]<br />
>user = example<br />
>pass = example<br />
>[run]<br />
>A = \\\\path\to\script.bat<br />

### Usage
<pre>
-m --map [options]    Map file share
-u --unmap [options]  Unmap file share
-r --run [options]    Run server batch file
-q --quiet            Do not show the logo
-i --info             Get system information
</pre>
